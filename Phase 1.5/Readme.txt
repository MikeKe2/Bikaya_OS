Generate the executable file for uarm using make and then using uarm for booting the kernel:
$ make uarm
$ uarm
From settings you need to select the "kernel" file generated by make uarm.

Generate the executable file for umps using make and then using umps for booting the kernel:
$ make umps
$ umps2
From setting you need to upload the "kernel.core.umps" and "kernel.stab.umps" files generated by make umps

Click "Power" to boot up the emulator.
Open terminal0 on "Terminals" or using ALT+0.
Click "Start" to boot the OS.
