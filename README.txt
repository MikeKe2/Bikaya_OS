per compilare il progetto lanciare da terminale il comando "make" seguito dal target di riferimento: uarm o umps
ES: make uarm/ make umps

Il progettò è organizzato con i file .c nella cartella principale, i file .h nella sottocartella include e i file specifici delle architetture nelle sottocartelle corrispondenti.
Nella cartella principale ci sono anche i terminali da usare sugli emulatori di arm e mps, i file stampante e i file di make necessari alla compilazione 

Per l'esecuizione sono necessari sia terminale che stampante, si possono sia usare quelli già presenti nella cartella che crearne di nuovi, attenzione ad abilitarli e configurarli nell'emulatore
